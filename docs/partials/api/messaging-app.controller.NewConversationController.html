NewConversationController
controller in module messaging-app

Description

Manages ./views/messages/new-conversation.html. This page allows the user to create a new conversation.

Dependencies

  • MessengerService

Methods

  • checkFields()

    Checks that the imageUrl and name fields are non-empty to enable the submit button.

  • createConversation()

    Adds the new conversation for the user using MessengerService.addConversation.

Properties

  • imageUrl

    URL of an image to represent other person in the new conversation.

  • name

    Name of the other person in the new conversation.
