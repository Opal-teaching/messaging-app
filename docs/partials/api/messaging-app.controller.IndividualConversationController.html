IndividualConversationController
controller in module messaging-app

Description

Manages ./views/messages/individual-conversation.html. This page shows the conversation with another user, and allows messages to be sent to them.

Dependencies

  • MessengerService

Methods

  • deleteConversation()

    Deletes the current conversation using MessengerService.deleteConversation.

  • initController()

    Initializes the controller and the scope variables.

  • sendMessage()

    Sends the typed message using MessengerService.sendMessage and reloads the conversation.

Properties

  • conversation

    Conversation currently being viewed.

  • messageContent

    Content of the message the user is typing.

  • noMessages

    Flag to toggle "No messages" text if the conversation is empty.
